# HudumaGuide TZ User Flows

## First-Time User

1. Opens app
2. Reads onboarding
3. Chooses to continue
4. Lands on Home
5. Sees search, quick actions, saved checklists, upcoming reminders, popular guides, and disclaimer

## Search for Government Service

1. User opens Home or Services
2. Types a Swahili or English term
3. App filters guides by title, keyword, category, summary, and common synonyms
4. User opens a guide
5. User reviews required documents and steps
6. User saves checklist or adds reminder
7. User can open official portal placeholder or report outdated info

## Service Checklist

1. User opens service guide
2. Taps "Hifadhi checklist"
3. Marks documents and steps as done
4. Readiness score updates
5. Checklist remains available offline from local persistence

## BiasharaStart Wizard

1. User opens Biashara tab
2. Taps "Start New Business Plan"
3. Answers setup questions
4. App recommends a simple roadmap
5. User saves the plan
6. User tracks setup progress from roadmap screen

## Create Reminder

1. User opens Reminders or a service guide
2. Taps create reminder
3. Enters title, category, date, repeat option, notes, notification preference
4. App validates fields and saves reminder locally
5. Reminder appears on Home and Reminders

## Report Outdated Information

1. User taps "Report outdated info" on a guide
2. User submits a note
3. Feedback is stored locally in MVP and mapped to `feedback_reports` in Supabase later
